# HERALD Intelligence

**Your AI journalist for VC secondaries.**

---

## What I can do

**Drop any link** — Spotify podcast, YouTube, TikTok, Twitter, Instagram, web article. I ingest it, extract what matters for VC secondaries, and surface the best editorial angles.

**Research live** — Type `/research [topic]` and I search the web for current deal activity, fund news, and market data.

**Build your newsletter** — I track topics throughout the week. On Friday I present the plan and wait for your approval before drafting. Type `/draft` to see what we have.

**Morning brief** — Type `/brief` to pull what is new from Elena Nisonoff, TBPN, and All-In Podcast.

**Manage your edition** — Type `/topics` to see what is saved. Tell me to "include this" and I save it. Tell me to "remove that" and I take it out.

**LinkedIn posts** — Type `/linkedin` or tell me to "repurpose this to LinkedIn" after any content.

---

## Commands

| Command | What it does |
|---------|-------------|
| `/research [topic]` | Live web search for VC secondaries intel |
| `/topics` | View this week's edition plan |
| `/brief` | Run morning ingestion from the three sources |
| `/draft` | Review topics and initiate newsletter draft |
| `/status` | System health and database stats |
| `/linkedin [topic]` | Generate a LinkedIn post |
| `/model` | Switch between AI models |
